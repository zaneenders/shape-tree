## Inline code and blocks

Use `CONTENT_PATH` to point at a content directory. A short Swift snippet:

```swift
let store = try ContentStore(contentDirectory: contentURL)
let groups = store.publishedPostGroups
```

## Blockquote

> Lorem ipsum dolor sit amet, consectetur adipiscing elit. Blockquotes sit on a soft gray panel and inherit the monospace rhythm of the page.

## Mixed body text

Nemo enim ipsam voluptatem quia voluptas sit aspernatur aut odit aut fugit. Inline `code spans` stay in the same family as body copy. **Bold** and *italic* pass through from markdown as expected.
