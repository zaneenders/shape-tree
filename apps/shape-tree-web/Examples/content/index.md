Lorem ipsum dolor sit amet, consectetur adipiscing elit. This demo site is built from markdown files on disk and served with [Lorikeet](https://github.com/zaneenders/lorikeet) and Hummingbird.

Set `CONTENT_PATH` to any folder of `.md` files. Subdirectories become navigation groups and index sections.

Start with the **Style Guide** for typography, links, lists, code, and blockquotes.
