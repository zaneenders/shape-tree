# Welcome to ShapeTree

This repository ships sample markdown under `content/` so you can run the web app without pointing at an external content directory.

## What you get

- **Articles** — public posts in `content/Articles/`
- **Favorites** — shorter notes in `content/Favorites/`
- **Front matter** — optional metadata between `-----` lines at the top of each file

Try opening this article in the Articles tab, then use the browser back button to return to the list.
